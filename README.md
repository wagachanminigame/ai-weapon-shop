## 🚀 デプロイ

このサイトはGitHub Pagesでホスティングされています。

### セットアップ方法

1. このリポジトリをフォーク
2. Settings > Pages > Source を `main` ブランチに設定
3. 自動的にデプロイされます！

### ローカル開発

```bash
# リポジトリをクローン
git clone https://github.com/wagachanminigame/ai-weapon-shop.git

# ディレクトリに移動
cd ai-weapon-shop

# ローカルサーバーを起動（Python）
python -m http.server 8000

# ブラウザで開く
# http://localhost:8000
```
